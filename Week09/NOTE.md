常规动态规划
	1.爬楼梯

	f(n) = f(n-1) + f(n-2)

	2.不同路径

	f(x, y) = f(x-1, y) + f(x, y-1)

	3.打家劫舍

	dp[i][0] = max{ dp[i-1][0], dp[i-1][1] } //不抢第i家
	dp[i][1] = dp[i-1][0] + nums[i] 

	4.最小路径和

	dp[i][j] = min{ dp[i-1][j], dp[i][j-1] } + nums[i][j]

	
高级动态规划
	复杂度来源：状态拥有更多维度、状态方程更复杂。
	举例：爬楼梯问题的深化
			每次可以爬 1、2、3 步
			每次可以爬 x[] 步
			前后不能走相同的步数
